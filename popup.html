⚡ SF Field Inspector

Hover over any field on a record page to inspect its metadata.

Active on Salesforce Lightning pages

What it shows

ALL API Name & Field Type
REF Lookup → parent object & relationship name
MD Master-Detail → parent object
FX Formula body (full expression)
$ Currency / Decimal precision & scale
PL Picklist active values
TXT Max length, Required, Unique flags
How to use: Navigate to any Salesforce record, then hover over a field label or value for ~0.4 seconds. The metadata panel appears automatically — click Copy API Name to copy it.
